## Download

Download the latest version of the Android app from [this link](https://raw.githubusercontent.com/TheGreatDemetrius/Game-Cities/master/source/game-cities.apk).

## Building

### Using Android Studio

Clone the repo, open it in Android Studio and build the app.

### Using command line

#### On Windows:

```powershell
.\gradlew.bat assembleDebug
```

#### On Linux/macOS:

```bash
chmod +x gradlew
./gradlew assembleDebug
```

*Note: You'll need Android SDK version 24.*
